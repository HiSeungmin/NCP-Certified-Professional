# NCP (NAVER Cloud Platform) 자격증 학습 정리

> NAVER Cloud Platform Certified Professional 취득을 목표로 한 학습 정리 저장소입니다.

## 📌 NCP란?

**NCP(NAVER Cloud Platform)** 는 네이버 클라우드에서 제공하는 퍼블릭 클라우드 서비스로, AWS·GCP와 유사하게 Compute, Storage, Network, Database, Media, Security, AI 등 다양한 클라우드 인프라 및 서비스를 제공합니다.

**NCP 기술자격증**은 네이버 클라우드 플랫폼을 활용하는 역량을 공식적으로 인증하는 자격으로, 난이도에 따라 3단계로 구성됩니다.

| 등급 | 명칭 | 과목 수 | 수준 |
|---|---|---|---|
| NCA | NAVER Cloud Platform Certified Associate | 1과목 | 입문/중급 |
| **NCP** | **NAVER Cloud Platform Certified Professional** | **3과목** | **전문가(중급)** |
| NCE | NAVER Cloud Platform Certified Expert | 4과목 | 전문가(상급) |

> 이 외에 AI 특화 자격증인 NCP-AI, NCE-AI 등도 존재합니다.

- 응시 자격 제한 없음 (학력·경력 무관)
- 2021년 6월부터 실기 시험은 폐지되어 **100% 온라인 필기 시험**으로 진행
- 과목별 60문제 / 60분, 과목당 응시료 5만원

## 🎯 목표: NCP (Professional) 취득

NCP 자격은 **3개 과목(200, 202, 207)을 모두 합격**해야 취득할 수 있습니다.

| 시험 코드 | 과목 | 주요 범위 |
|---|---|---|
| **200** | Overview, Compute / Storage | 플랫폼 개요, Server/Storage 상품, 컴퓨팅 인프라 |
| **202** | Network / Media, Database / Management / Analytics | LB, CDN, VPC, Cloud DB, 미디어, 관리/분석 상품 |
| **207** | Troubleshooting | 인프라 장애 진단·해결, Linux/Windows 명령어, 네트워크 트러블슈팅 |
